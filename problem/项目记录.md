# Obsidian 笔记网站发布记录

## 项目概述

将三个 Obsidian vault 发布为静态网站，部署到阿里云轻量级 Ubuntu 服务器，通过 IP 访问，支持自动同步更新。

## 技术栈

| 层级     | 技术                          |
| ------ | --------------------------- |
| 静态网站生成 | Quartz v4.5.2               |
| 托管     | 阿里云轻量级 Ubuntu + nginx       |
| 自动部署   | GitHub Actions（每 15 分钟自动构建） |
| 内容源    | 3 个私有 GitHub 仓库             |

## 本地环境

| 项目        | 路径                                                      |
| --------- | ------------------------------------------------------- |
| Quartz 项目 | `F:\GitHub_Repository\obsidian-publish\`                |
| 数据结构笔记    | `F:\GitHub_Repository\Data_Struct\` (GitHub: `DS_note`) |
| 数学笔记      | `F:\GitHub_Repository\Math_note\` (GitHub: `Math_note`) |
| 英语笔记      | `F:\GitHub_Repository\En_learn\` (GitHub: `En_word`)    |

## 网站地址

**http://47.108.63.224/**

## 同步机制

```
Obsidian 编辑笔记 → obsidian-git 自动 commit + push 到 GitHub
       ↓
GitHub Actions 每 15 分钟自动执行
       ↓
克隆三个 vault 私有仓库 → Quartz 构建 → rsync 部署到阿里云
       ↓
网站自动更新
```

## 已知注意事项

1. **图片路径**：`quartz.config.ts` 中 `markdownLinkResolution` 必须设为 `"relative"`，否则图片相对路径会被错误转换成 `../../` 形式
2. **链接层级**：同步脚本会去掉 vault 的顶层包装目录（如 `DS/`、`Math/`），首页链接需与之一致
3. **私有仓库**：GitHub Actions 克隆私有仓库需要在 URL 中嵌入 PAT（`https://oauth2:$GH_TOKEN@github.com/...`）
4. **字体**：使用系统字体 `Microsoft YaHei`，避免 Google Fonts 在国内加载失败
5. **OG 图片**：`CustomOgImages` 插件已禁用，因 Windows 系统缺少所需字体
6. **LaTeX 警告**：笔记中 `$...$` 内使用了中文字符，KaTeX 会给出警告但不影响渲染
